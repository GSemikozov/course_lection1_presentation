Сброс стилей

Reset, normalize

Нужны для сброса браузерных стилей до общего состояния.

Reset - это не конкретный файл, а набор правил. Можно составить самому или взять из open source. Подход предложил Эрик Мейер - его reset.css.

Normalize - современная альтернатива. Подход предложил Николас Галлахер. Нормализовать стили — значит убрать различия отображения в разных браузерах, а не "обнулить" до отправной точки как в случае с reset.css. Живет тут.

Блок Элемент Модификатор

Компонентный подход к веб-разработке.
Позволяет писать переиспользуемый css, в основе - разделение на блоки.

Системы организации кода. БЭМ

Блок - функционально независимый компонент страницы, который может быть повторно использован.

  • Не надо задавать внешние отступы и положение
  • Можно вкладывать друг в друга

.menu .header .footer .bell .card

Системы организации кода. БЭМ

Элемент - составная часть блока, которая не может использоваться без него.

  • Всегда часть блока, а не другого элемента
  • Можно вкладывать друг в друга

.menu__item .header__copyright .footer__row .bell__mark

Системы организации кода. БЭМ

Модификатор - внешний вид, состояние или поведение блока либо элемента.

  • Нельзя использовать самостоятельно, только с модифицируемым блоком или элементом

.menu_disabled .footer__row_direction-top .card_size-single

Как "миксовать" блоки (например, выравнивать по сетке)?

  • Блоки всегда встраиваются в другие блоки
  • Внешнее поведение дочерних блоков всегда определяется родителем
  • Поэтому можно использовать элемент родительского блока рядом с классом блока, которому нужно задать определенное поведение (выравнивание в общей сетке и пр.)

Семантическая верстка. Details & Summary

              summary {
                display: block;  // Для Firefox
              }
              summary::-webkit-details-marker {
                display: none;  // Для Webkit и Safari
              }
              summary .marker {
                display: inline-block;
                padding-right: 0.5em;
              }
              details[open] .marker {
                transform: rotate(90deg);
              }

CSS-переменные

CSS-переменные

Имена свойств с префиксом --, например --important-property. Свойства затем можно использовать в документе с помощью функции var(). 

                :root {
                  --primary-color: red;  // Определение
                }
            
                .block {
                  color: var(--primary-color, red);  // Использование
                }

CSS-переменные. Переопределение

                        .one {
                          --text-color: blue;
                        }
                        .two {
                          --text-color: red;
                          color: var(--text-color);
                        }
                    
                        <div class="one">
                          <div class="two">
                            Какого цвета текст?
                          </div>
                        </div>
